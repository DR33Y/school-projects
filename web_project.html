variables typs:

  1. integer - Stores whole numbers without decimal points, like 5, -10, or 0.
  2. float - Stores decimal numbers, like 3.14, -0.5, or 2.0.
  3. string - Stores sequences of characters, enclosed in single or double quotes, like "hello" or 'Python'.
  4. char - In Python, characters are represented as strings of length 1. For example, 'a', 'X', or '&'. However, Python doesn't have a separate char type like some other languages.
  5. boolean - Represents truth values, either True or False.

What is if else statement in python:
In Python, the if else statement is used for decision-making. It allows you to execute different blocks of code based on whether a condition is true or false.
If the condition specified after if is true, the code block under if is executed; otherwise, the code block under else is executed. It provides a way to choose between two different courses of action based on a condition.
